## Overview

This project is a multi-page personal portfolio website developed as part of the Thiranex Web Development Internship Task 1.

The website is built using semantic HTML5 and CSS, following accessibility and responsive design best practices. It showcases personal information, skills, projects, and contact details in a professional format.
