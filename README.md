# Thiranex Task 1 - Personal Portfolio Website

## Overview

This project is a multi-page personal portfolio website developed as part of the Thiranex Web Development Internship Task 1.

The website is built using semantic HTML5 and CSS, following accessibility and responsive design best practices. It showcases personal information, skills, projects, and contact details in a professional format.

## Features

* Multi-page website structure
* Semantic HTML5 elements
* Responsive design
* Accessibility-friendly navigation
* SEO meta tags
* About Me section
* Projects showcase
* Contact page
* Clean and professional user interface

## Pages

* Home (index.html)
* About (about.html)
* Projects (projects.html)
* Contact (contact.html)

## Technologies Used

* HTML5
* CSS3
